# Project Overview

- Purpose: AI-Assisted Workflows platform combining CLI workflows, Python analysis tools, and multi-agent orchestration for code quality management.
- Key Components: Claude Code and Codex CLI configurations; shared Python analyzers; test codebases for validation.
- Architecture: BaseAnalyzer framework, 8-agent orchestration, integration with GitHub Actions, Serena MCP search, and multi-language LSP support.
- Tech Stack: Python 3.11+, TypeScript/JavaScript, Bash, SQL; tooling includes Semgrep, Detect-secrets, Lizard, Ruff, JSCPD, custom frameworks.
- Repo Structure: `systems/` for CLI workflows, `shared/` for analyzers and infrastructure, `test_codebase/` for validation apps, `docs/` and `todos/` for documentation and task tracking.
