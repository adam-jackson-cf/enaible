## Shared Skills

When you see the below skills mentioned use the referenced workflow skill file.

| Skill             | Path                                    | Semantic trigger                                                   |
| ----------------- | --------------------------------------- | ------------------------------------------------------------------ |
| codify-pr-reviews | `.pi/skills/codify-pr-reviews/SKILL.md` | User wants to turn PR review feedback into rules or reduce repeats |
| docs-scraper      | `.pi/skills/docs-scraper/SKILL.md`      | User asks to scrape/save docs from URLs into markdown              |
| research          | `shared/skills/research/SKILL.md`       | User wants the agent to perform deep research with source capture  |

## Network access

If a user request, skill or workflow implies or requires network access for a given task i.e. web search etc - you can consider your bash access to curl a tool that meets such requirements, use it to meet the task goal.
